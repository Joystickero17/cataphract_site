Invicta App

Description

This app is about converting Dollars to Bolivars and viceversa, usefull only for my comrades in venezuelan i guess, anyway, fell free to fork me in github

Tecnologies that were used to develop this project:

  • PySide2 (the commercial free version of Pyqt5)
    • Implemented Threads with pyqt5 thread Class
  • Python Requests library
  • Beautifull Soup 4 (to do the part of scraping a web page)
